# Karel-Sala-KaLA

**KaLa Automobile** – Autoankauf Website für Karel Sala, Nürnberg/Eckental.

## Tech-Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- TypeScript
- Full SEO (Meta, Geo, JSON-LD, robots.txt, sitemap)

## Starten

```bash
npm install
npm run dev
```
